## New .env Format

**Before (complex):**
```
LINKEDIN_COOKIES=[{"name":"li_at","value":"xxx","domain":".linkedin.com"},...]
```

**After (simple):**
```
LINKEDIN_LI_AT=AQEDATxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## How to Get li_at Cookie

1. Open LinkedIn in your browser and login
2. Press F12 → Application tab → Cookies → linkedin.com
3. Find the `li_at` cookie and copy its **Value**
4. Paste into `LINKEDIN_LI_AT=` in your .env file
